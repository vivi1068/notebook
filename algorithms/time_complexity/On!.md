`O(n!)`被称为阶乘阶复杂度, 
穷尽求所有排列组合的算法通常都是阶乘阶复杂度, 比如说下面例子, 
当 n 在递增时, 执行次数成几何倍数的增长, 
这种情况被称为阶乘阶复杂度.
```shell
2! = 2 x 1 = 2                                      # 2的阶乘有2个排列组合
3! = 3 x 2 x 1 = 6                                  # 3的阶乘有6个排列组合
4! = 4 x 3 x 2 x 1 = 24                             # 4的阶乘有24个排列组合
5! = 5 x 4 x 3 x 2 x 1 = 120                        # 5的阶乘有120个排列组合
6! = 6 x 5 x 4 x 3 x 2 x 1 = 720                    # 6的阶乘有720个排列组合
7! = 7 x 6 x 5 x 4 x 3 x 2 x 1 = 5.040              # 7的阶乘有5040个排列组合
8! = 8 x 7 x 6 x 5 x 4 x 3 x 2 x 1 = 40.320         # 8的阶乘有40320个排列组合
```

def heap_permutation(data, n):
    if n == 1:
        print(data)
        return
    
    for i in range(n):
        heap_permutation(data, n - 1)
        if n % 2 == 0:
            data[i], data[n-1] = data[n-1], data[i]
        else:
            data[0], data[n-1] = data[n-1], data[0]

data = [1, 2, 3, 4]
heap_permutation(data, len(data))
